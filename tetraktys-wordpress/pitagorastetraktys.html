Publicaciones Académicas con Polymer y Google

[one-third-first]

Hola. Este método que hemos llamado PiTaGoRAS:

Polymer Internet Text for Academy with Google Running A Server

Te enseña a crear publicaciones en formato académico para distribución web, usando Google Cloud Platform.

  1. PiTAGoRAS utiliza lenguajes de marcado: Markdown .md, HTML .html y LaTeX .txt. Ingrese a cada vinculo y conozca mas de su estructura.
[/one-third-first] [two-thirds]

  1. Ingrese al SSH de su MV y ejecute los comandos de linux para instalar la estructura de polymer-pub.

Intalar nodejs

$ curl -sL https://deb.nodesource.com/setup_9.x | sudo -E bash -
$ sudo apt-get install -y nodejs

Compilar e instalar extensiones nativas

$ sudo apt-get install -y build-essential

Instalar globalmente npm

$ mkdir ~/.npm-global
$ npm config set prefix '~/.npm-global'
$ sudo nano ~/.profile

Pegar

$ export PATH=~/.npm-global/bin:$PATH

Guardar y volver a actualizar con el comando

$ source ~/.profile

probar

$ npm install -g jshint

Instalar y comprobar requisitos

$ node --version
$ npm install npm@latest -g
$ git --version
$ npm install -g bower

Instalar polymer CLI

$ npm install -g polymer-cli

Actualizar permisos

$ sudo chmod 777 htdocs
$ cd htdocs && sudo chmod 777 index.html

Instalar proyecto

$ cd && cd htdocs && polymer init
$ ❯ polymer-2-application - A simple Polymer 2.0 application
nombre: polymer-pub
$ polymer serve
cierre el SSH, ingrese de nuevo y ejecute cada línea de comando a continuación para crear la estructura del proyecto.

  1. Descarge e Instale una solución para protocolo de transferencia de archivos, recomedamos FileZilla
  1. Conecte su MV con el FTP, con Filezilla es muy simple:
  • Descarge la clave en archivo .PEM o .PPK
  • En Filezilla vaya a edición/opciones/sftp
  • Adicione la clave que descargo
  • En Filezilla vaya a archivos/gestor de sitios/sftp
  • Asegurese de enlazar la IP de Bitnami a su dominio DNS: Tipo A, cree un sitio: bitnami@tudominio.com, usuario: bitnami y elija el protocolo SFTP e ingrese a su sitio.
  • Asi debe ver su estructura de proyecto polymer-pub.
  1. Ahora vaya al navegador e ingrese su dominio o IP para ingresar al escritorio de PiTAGoRaS. Ingrese a www.panaceapp.site y conozca el DEMO
